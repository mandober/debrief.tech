# Semantic keys

Index of semantic keys
- buffer    (b)
- change    (cC sS dD)
- insert    (iI aA oO rR)
- marks     (m ' `)
- registers (")
- macros    (q @)



```
m      Marks
mm     Leave a mark 'm'
'm     Jump to line with mark 'm'
`m     Jump to mark 'm'


"      Registers


@      Macros
qq     Start recording a macro to slot 'q'
qQ     Start recording a macro by prepending to slot 'q'
```
