# Linux shells


Linux shells
- Bourne shell, `sh`
- Bourne-again shell, `bash`
- Z-shell, `zsh`
- Friendly Interactive Shell, `fish`
- Korn sell, `ksh`
- C-shell, `csh`
- TENEX C Shell, `tcsh`
- Thompson shell, `tsh`
- PowerShell, `pwsh`
- Almquist shell, `ash`
- Debian Almquist Shell, `dash`
- Free version of Korn Shell, `pdksh`
- The AT&T Plan 9 shell ported to Unix, `rc`
- The AT&T KornShell, `ksh93u+m`
- Scheme shell, `scsh`
