# Shell :: History

## Abbridged History

sh by Ken Thomson
- Year: 1971
- CLI
- I/O redirection
- but no variables or pipelines

Bourne Shell by Stephen Bourne
- Year: 1979
- advanced control flows
- shell variables
- environemnt variables
- signal handling

csh by Bill Joy
- Year: 1979
- C-like syntax
- aliases
- command history
- job control
- tilde (~)

bash by Brian Fox
- Year: 1989
- command-line completion
- brace extension
- traps
- here-doc
