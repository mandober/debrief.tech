# Bash completions


compgen [-abcdefgjksuv]

```bash
compgen -a # aliases
compgen -b # builtins
compgen -c # commands
compgen -d # dir
compgen -e # exported vars (declare -x)
compgen -f # functions
compgen -g # groups
compgen -j # jobs
compgen -k # 
compgen -s # services
compgen -u # users
compgen -v # vars
```
