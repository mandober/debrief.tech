# Powershell

- name: Powershell
- file extensions:
  - script: `.ps1`
  - module: `.psm`, `.psm1`
  - console: `.psc1`
  - `.pssc`
  - `.ps1xml`
  - desired conf: `.psd1`

- purpose: DSL, shell scripting
- implementation: relies on .NET's CLR
- case sensitivity: *case insensitive*
- file encosing: UTF-8
- line endings: CR+LF
- directory separator: recognizes both, win (`\`) and unix (`/`)


Features
- objects instead of text

Type system
- dynamic, optional static typing
