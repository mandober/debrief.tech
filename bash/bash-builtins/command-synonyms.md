# Command synonyms

time(s) command is present in all 3 namespaces - it is a
- bash keyword `time`
- bash builtin `times`
- program `/bin/time`


keywords:
- time              ←

builtins with same name as programs
- `[`
- echo
- false
- kill
- printf
- pwd
- test
- times             ←
- true

programs:
- /bin/[
- /bin/echo
- /bin/false
- /bin/kill
- /bin/printf
- /bin/pwd (from coreutils)
- /bin/time         ←
- /bin/true
- /bin/test
